Tu es KHEDIM AI, expert en développement logiciel. Génère un projet complet appelé "TACTI-BALL PRO" — une plateforme d'analyse tactique football IA.

== STACK TECHNIQUE ==
- Backend : Python FastAPI + Groq (modèles llama-3.3-70b-versatile, llama-4-scout, qwen3-32b)
- Frontend : HTML5 + CSS3 + JavaScript vanilla (SPA — un seul fichier index.html)
- Pas de base de données requise
- Déployable gratuitement sur Render.com (backend) + InfinityFree (frontend)

== FONCTIONNALITÉS OBLIGATOIRES ==

1. ANALYSE VIDÉO (2 modes) :
   - Upload fichier local (MP4, AVI, MOV, MKV, WebM — max 500MB)
   - Analyse depuis URL (YouTube, YouTube Live, liens directs)
   - L'IA analyse : formations détectées, réseau de passes, zones de pressing, statistiques
   - Retourne données joueurs en JSON pour affichage sur terrain

2. TERRAIN 3D INTERACTIF :
   - Canvas HTML5 avec perspective 3D (vue trapèze — pas de WebGL)
   - Joueurs représentés par cercles colorés avec numéro
   - Ballon qui se déplace en mouvement temps réel (animation requestAnimationFrame)
   - Joueurs qui se déplacent selon leur formation avec mouvement naturel
   - Modes : Vue 3D perspective / Vue Top / Vue Côté
   - Heatmap par équipe (zones de chaleur colorées)
   - Réseau de passes affiché en lignes translucides
   - Formations supportées : 4-3-3, 4-4-2, 4-2-3-1, 3-5-2, 5-3-2, 4-5-1
   - Couleurs maillot personnalisables (color picker)

3. LIVE MATCH EN DIRECT (2 sources) :
   - Source caméra locale : getUserMedia() → capture frames → envoi base64 API
   - Source URL stream : YouTube Live ou lien direct
   - OCR via Groq Vision : détecte numéros maillots, couleurs équipes, score à l'écran, chronomètre, position ballon
   - Fréquence d'analyse configurable (500ms à 5000ms)
   - WebSocket /ws/live pour mises à jour temps réel

4. DASHBOARD STATISTIQUES COMPLET :
   - Tableau de score (scoreboard) avec badge LIVE animé
   - Barre de possession (%) animée pour les 2 équipes
   - Stats complètes : tirs, tirs cadrés, passes, fautes, corners, hors-jeu, cartons jaunes/rouges
   - Journal d'événements avec icônes (⚽ 🟨 🟥 🎯 🚩)
   - Boutons pour ajouter événements manuellement (but, carton, tir, corner, faute)
   - Mini terrain avec positions joueurs mises à jour en temps réel

5. ANALYSE TACTIQUE IA :
   - Analyse formation vs adversaire
   - Plan de jeu (défense, transition, attaque)
   - Rapport de match complet
   - Note tactique /10

6. COACH IA (TACTI-BOT) :
   - Chat conversationnel avec historique
   - Expert tactiques, formations, règles FIFA/UEFA
   - Boutons raccourcis pour questions fréquentes

7. PLAN D'ENTRAÎNEMENT :
   - Programme semaine par semaine
   - Exercices spécifiques au focus choisi
   - Analyse joueur individuelle
